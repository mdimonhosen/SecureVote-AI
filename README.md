# SecureVote AI – Face Recognition Voting System.


# Description
SecureVote AI is a mobile-based voting system that uses facial recognition technology to provide a secure, reliable, and convenient remote voting experience. It ensures one-person-one-vote through biometric authentication, reduces fraud, and improves transparency while maintaining user privacy and accessibility.


# Features
- Secure user registration with identity verification
- Facial recognition authentication with liveness detection
- Remote voting from any location
- End-to-end encryption for vote security
- Real-time monitoring dashboard for administrators
- Digital receipt generation while maintaining anonymity
- Integration with cloud services for authentication and data storage


# Technologies Used
- Frontend: Flutter (Dart)
- Backend: Python (FastAPI)
- AI/ML: TensorFlow Lite, Google ML Kit
- Database: PostgreSQL, Firebase, Supabase
- Development Tools: Android Studio, VS Code, Git


# System Architecture
The system consists of the following components:
- Mobile application built with Flutter
- Backend API developed using FastAPI
- AI modules for facial recognition and liveness detection
- Cloud services for authentication and real-time data synchronization
- Database for secure storage of voter records


# Installation

Clone the repository:
- git clone https://github.com/mdimonhosen/SecureVote-AI.git
- cd securevote-ai

Frontend setup (Flutter):
- flutter pub get
- flutter run

- Backend setup (FastAPI):
- pip install -r requirements.txt
- uvicorn main:app --reload
